Workflow 10 Steps to Feature-First

Guia de Migração — Passo a Passo

Siga a ordem estrita para evitar regressão

01

Criar Estrutura

Crie a pasta da feature e subdiretórios padrão.

mkdir src/features/xy
02

Extrair API

Mova chamadas HTTP diretas para camada de API.

api/myFeatureApi.ts
03

Hooks (Server)

Implemente React Query para cache e loading states.

hooks/useMyQuery.ts
04

Store (Client)

Defina estado global de UI com Zustand se necessário.

stores/useMyStore.ts
05

Validators

Isole regras de validação em funções puras.

validators/schema.ts
06

ViewModels

Orquestre a lógica conectando Store, Hooks e API.

viewModels/useLogic.ts
07

Refatorar Telas

Limpe os componentes de UI (Dumb Components).

screens/MyScreen.tsx
08

Public API

Defina o contrato público de exportação no index.

index.ts
09

Atualizar Imports

Corrija referências no projeto para a nova feature.

Fix references
10

Remover Legado

Delete os arquivos antigos sem piedade.

rm src/legacy/...
Processo Padronizado v1.0
